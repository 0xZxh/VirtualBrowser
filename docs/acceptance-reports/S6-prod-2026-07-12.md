## 验证记录

### 检查项 1 — GET /health（mongo 模式）

- **命令：**
  ```powershell
  curl.exe -s http://127.0.0.1:3001/health
  ```
- **预期：** `ok: true`，`storage: mongo`，`mongo: connected`
- **实际：**
  ```json
  {"ok":true,"service":"virtualbrowser-backend","stack":"nestjs","storage":"mongo","mongo":"connected"}
  ```
- **结果：** ☑ 通过

### 检查项 2 — POST /auth/login（admin/admin123）

- **命令：**
  ```powershell
  curl.exe -s -X POST http://127.0.0.1:3001/auth/login `
    -H "Content-Type: application/json" `
    --data-binary "@$env:TEMP\vb-login.json"
  # vb-login.json 内容: {"username":"admin","password":"admin123"}
  ```
- **预期：** `code=0`，`data.token` 非空
- **实际：**
  ```json
  {"code":0,"data":{"token":"e7dbfdb64e1be2276ad2451e21e36def4f49886377ef5902","user":{"id":"6a53a21909e11055e69cd9d2","username":"admin","name":"管理员","roles":["admin"],"tenantId":"1"}}}
  ```
- **结果：** ☑ 通过

### 检查项 3 — POST /api/environments 创建环境

- **命令：**
  ```powershell
  curl.exe -s -w "`nHTTP_STATUS:%{http_code}" -X POST http://127.0.0.1:3001/api/environments `
    -H "Content-Type: application/json" `
    -H "Authorization: Bearer <TOKEN>" `
    --data-binary "@$env:TEMP\vb-env.json"
  # vb-env.json 内容: {"name":"S6-prod-final","group":"default"}
  ```
- **预期：** HTTP 2xx，返回新建环境
- **实际：**
  ```json
  {"code":0,"data":{"id":2,"name":"S6-prod-final","group":"default","ownerId":"6a53a21909e11055e69cd9d2","tenantId":"1"}}
  HTTP_STATUS:201
  ```
- **结果：** ☑ 通过（201 Created）

### 检查项 4 — CORS 白名单

- **命令：**
  ```powershell
  curl.exe -s -D - -o NUL `
    -H "Origin: http://localhost:9527" `
    -H "Access-Control-Request-Method: POST" `
    -X OPTIONS http://127.0.0.1:3001/auth/login
  ```
- **预期：** `Access-Control-Allow-Origin: http://localhost:9527`
- **实际：**
  ```
  HTTP/1.1 204 No Content
  Access-Control-Allow-Origin: http://localhost:9527
  Access-Control-Allow-Credentials: true
  Access-Control-Allow-Methods: GET,HEAD,PUT,PATCH,POST,DELETE
  ```
- **结果：** ☑ 通过
