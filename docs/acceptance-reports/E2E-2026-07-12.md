### 检查项 4 — Compat API 全链：addBrowser → launchBrowser → stopBrowser（S8）

- **命令：**
  ```powershell
  # addBrowser
  curl -X POST http://127.0.0.1:9000/api/addBrowser -H "Content-Type: application/json" -d "{\"name\":\"e2e-compat\",\"group\":\"默认分组\"}"
  # launchBrowser
  curl -X POST http://127.0.0.1:9000/api/launchBrowser -H "Content-Type: application/json" -d "{\"id\":4}"
  # stopBrowser
  curl -X POST http://127.0.0.1:9000/api/stopBrowser -H "Content-Type: application/json" -d "{\"id\":4}"
  ```
- **预期：** 创建 id=4 → launch 返回 `debuggingPort` → stop 成功
- **实际：**
  ```json
  addBrowser → {"success":true,"data":{"id":4}}
  launchBrowser → {"success":true,"data":{"debuggingPort":19203}}
  stopBrowser → {"success":true}
  getBrowserRunningList → 含 id=4，stop 后进程退出
  ```
- **备注：** 对历史 env `id=1` 首次 launch 报 `CDP 端口 19200 未就绪`；新建 env `id=4` 正常。`automation/test-api.js` 对 id=1 同样失败。
- **结果：** ☑ 通过（新建环境链路）；☐ id=1 历史环境 launch 待排查
