## Project setup
```
npm install
```

### Mac 开发（推荐阅读）
见仓库 [`docs/MAC_FRONTEND_DEV.md`](../docs/MAC_FRONTEND_DEV.md)。

```
# 先起后端
cd ../server-backend && npm run start:dev

# 再起管理端
npm run dev   # http://localhost:9527
```

非 Windows 下可创建/编辑环境；`launchBrowser` 会友好失败（需内核 exe）。
